About FraudGuard AI

Advanced Credit Card Fraud Detection System powered by Machine Learning and Explainable AI

Project Overview

FraudGuard AI is a comprehensive fraud detection system developed as part of an MSc Data Science project. It leverages state-of-the-art machine learning algorithms and explainable AI techniques to detect fraudulent credit card transactions in real-time.

The system is built on the famous ULB Credit Card Fraud Detection dataset from Kaggle, which contains 284,807 transactions with only 0.172% fraudulent cases, presenting a challenging class imbalance problem.

Key Features

  • Multiple ML models (XGBoost, Random Forest, LightGBM, etc.)
  • Explainable AI with SHAP and LIME
  • Real-time fraud detection
  • Interactive web dashboard
  • Advanced imbalance handling
  • Comprehensive model comparison

Explainable AI Techniques

SHAP (SHapley Additive exPlanations)

Provides unified measure of feature importance by calculating the contribution of each feature to the prediction based on game theory.

LIME (Local Interpretable Model-agnostic Explanations)

Explains individual predictions by approximating the model locally with an interpretable model around the prediction.

Dataset Information

ULB Credit Card Fraud Detection Dataset
  • Total Transactions: 284,807
  • Fraudulent Transactions: 492 (0.172%)
  • Features: 30 (28 PCA + Time + Amount)
  • Time Period: 2 days in September 2013

  • Severe class imbalance (0.172% fraud)
  • Anonymized features (PCA transformed)

  • Real-world transaction patterns
